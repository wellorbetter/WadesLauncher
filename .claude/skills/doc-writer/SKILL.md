# /doc-writer 技术文档生成

自动分析代码并生成结构化技术文档。

## 触发方式
```
/doc-writer <模块路径或功能名>
/doc-writer <模块路径> --type=api       # API 文档
/doc-writer <模块路径> --type=arch      # 架构文档
/doc-writer <模块路径> --type=share     # 技术分享文档
```

## 文档结构模板

### 技术文档（默认）
1. **背景与目标**: 为什么做，解决什么问题
2. **架构概览**: Mermaid 架构图 + 模块说明
3. **核心流程**: 时序图 + 流程图 + 关键代码路径
4. **设计决策**: 为什么选这个方案，考虑过哪些替代方案
5. **架构模式与设计模式**: 本功能使用的模式，附参考链接和简要说明
6. **API 文档**: 接口定义、参数、返回值、错误码
7. **注意事项**: 已知限制、性能考量、安全注意点

### API 文档
- RESTful 接口：Method + Path + Request/Response + 错误码
- gRPC：Proto 定义 + 使用示例
- FFI：Rust 函数签名 + 调用约定 + 内存管理

### 架构文档
- 系统全景图 + 模块依赖图 + 数据流图
- 技术选型理由 + 架构模式说明

## 输出位置
`docs/<feature-name>-tech-doc.md`
