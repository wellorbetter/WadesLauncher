# /diagram 专业绘图

独立的 Mermaid 图表生成 Skill，三种工作模式。

## 触发方式
```
/diagram <描述>                    # 从描述生成
/diagram <文件路径> 的类关系        # 从代码分析
/diagram 展示 <模块> 的完整技术架构  # 技术展示（组合多张图）
```

## 支持的图表类型

| 图表类型 | 适用场景 | Mermaid 语法 |
|---------|---------|-------------|
| 架构图 | 系统结构、模块关系 | graph TD / graph LR |
| 类图 | 继承/组合/依赖关系 | classDiagram |
| 时序图 | 多角色交互时间顺序 | sequenceDiagram |
| 流程图 | 步骤和分支判断 | flowchart TD |
| 状态图 | 生命周期状态变化 | stateDiagram-v2 |
| ER图 | 数据库实体关系 | erDiagram |

## 配色方案（蓝青色系）

| 颜色 | 色值 | 语义 |
|------|------|------|
| 蓝色 | #4A90D9 | 核心流程 / 主要模块 |
| 深蓝 | #2D6CB4 | 边框 / 强调 |
| 青色 | #67B7DC | 辅助模块 / 次要流程 |
| 浅蓝 | #E8F4FD | 背景 / 分组 |
| 黄色 | #FFF8E1 | 注释 / 说明 |
| 红色 | #E57373 | 错误 / 异常路径 |
| 绿色 | #81C784 | 成功 / 完成状态 |

## 质量检查清单
每张图生成后自动检查：
- [ ] 包含 `%%{init}%%` 主题配置
- [ ] 中文标注且 ≤15 字
- [ ] 关键路径粗线标注
- [ ] 异常路径虚线 + 红色
- [ ] 节点数 ≤15（超过则拆图）
- [ ] subgraph 分组逻辑清晰
- [ ] 连线标签简洁（≤8 字）

## 语义符号
- 📱 移动端/客户端  🖥️ 服务端/后台  💾 数据库/存储
- 🔌 接口/API  🔀 分支/判断  🔄 循环/重试
- 🔒 安全/权限  ✅ 成功  ❌ 失败  ⚠️ 警告
